## ¿Qué es el Analizador de Llamadas Pro?

El **Analizador de Llamadas Pro** (Coyprot Analysis) es una herramienta de escritorio para análisis forense de registros de detalle de llamadas (CDR). Permite a investigadores y analistas de seguridad procesar sábanas de llamadas en formato Excel o CSV, geolocalizar antenas celulares y generar informes HTML interactivos listos para compartir.

<Info>
  Esta herramienta fue desarrollada para **Coyprot**, empresa especializada en análisis de seguridad e inteligencia.
</Info>

## Capacidades principales

<CardGroup cols={3}>
  <Card title="Procesamiento de datos" icon="table">
    Carga sábanas Excel/CSV con mapeo inteligente de columnas y normalización automática de números telefónicos.
  </Card>
  <Card title="Geocodificación multinivel" icon="location-dot">
    GPS nativo, base de datos de celdas y fallback por inferencia de municipio colombiano.
  </Card>
  <Card title="Mapas interactivos" icon="map">
    Genera mapas de clusters, rutas cronológicas y mapas de calor con Folium.
  </Card>
  <Card title="Dashboard estadístico" icon="chart-bar">
    Gráficos animados de tendencias horarias, Top 10 y distribución de ubicaciones.
  </Card>
  <Card title="Gestión documental" icon="file-pdf">
    Adjunta PDFs categorizados (Financiero, Judicial, Vehículos, etc.) al informe final.
  </Card>
  <Card title="Exportación y FTP" icon="upload">
    Genera informes HTML portables y súbelos automáticamente por FTP.
  </Card>
</CardGroup>

## Flujo de trabajo

<Steps>
  <Step title="Carga la sábana de llamadas">
    Selecciona tu archivo Excel o CSV con el historial de llamadas del objetivo.
  </Step>
  <Step title="Mapea las columnas">
    El asistente gráfico te ayuda a relacionar cada columna con los campos estándar del sistema.
  </Step>
  <Step title="Enriquece los datos">
    Asigna alias a números clave, agrega datos del caso y adjunta documentos PDF.
  </Step>
  <Step title="Genera y exporta">
    Con un clic, el sistema produce un informe HTML completo con mapas, gráficos y documentos adjuntos.
  </Step>
</Steps>

## Modos de ejecución

<Tabs>
  <Tab title="Interfaz Gráfica (recomendado)">
    Inicia la aplicación con ventanas visuales:

    ```bash
    python src/gui.py
    ```

    La interfaz Tkinter ofrece un flujo guiado paso a paso para cargar datos, asignar nombres, gestionar adjuntos y exportar el informe.
  </Tab>
  <Tab title="Línea de Comandos">
    Para ejecuciones automatizadas o integración en scripts:

    ```bash
    python run.py
    ```

    También puedes ejecutar directamente desde `src/`:

    ```bash
    python src/main.py
    ```
  </Tab>
</Tabs>
