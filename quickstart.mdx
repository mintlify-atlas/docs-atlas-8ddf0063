title: "Inicio Rápido"
description: "Pon en marcha el Analizador de Llamadas Pro y genera tu primer informe desde cero."
---

Esta guía te lleva desde cero hasta tener un informe HTML generado. El proceso completo toma menos de cinco minutos en una máquina con Python ya instalado.

<Steps>
  <Step title="Clonar el repositorio">
    Descarga el código fuente en tu equipo:

    ```bash
    git clone https://github.com/coyprot/coyprot_analisis_sabanas.git
    cd coyprot_analisis_sabanas
    ```
  </Step>

  <Step title="Crear el entorno virtual">
    Crea un entorno virtual para aislar las dependencias del proyecto:

    <Tabs>
      <Tab title="Windows">
        ```bat
        python -m venv venv
        venv\Scripts\activate
        ```
      </Tab>
      <Tab title="macOS / Linux">
        ```bash
        python3 -m venv venv
        source venv/bin/activate
        ```
      </Tab>
    </Tabs>

    Cuando el entorno está activo, el prompt del terminal muestra el prefijo `(venv)`.
  </Step>

  <Step title="Instalar dependencias">
    Instala todas las librerías listadas en `requirements.txt`:

    ```bash
    pip install -r requirements.txt
    ```

    Esto descarga e instala pandas, folium, matplotlib, jinja2 y el resto de dependencias. Puede tardar uno o dos minutos según la velocidad de la conexión.
  </Step>

  <Step title="Configurar las variables de entorno (opcional)">
    Si vas a usar la subida FTP, copia el archivo de ejemplo y rellena tus credenciales:

    ```bash
    cp .env.example .env
    ```

    Edita `.env` con un editor de texto:

    ```ini
    FTP_HOST=tu_dominio.com
    FTP_USER=tu_usuario_ftp
    FTP_PASS=tu_contraseña_ftp
    FTP_PUBLIC_HTML=public_html
    LOG_LEVEL=INFO
    GEO_PROXIMITY_KM=50
    ```

    <Note>
      Si no vas a subir informes a FTP, puedes omitir este paso por completo.
    </Note>
  </Step>

  <Step title="Iniciar la interfaz gráfica">
    Lanza la aplicación desde la raíz del proyecto:

    ```bash
    python run.py
    ```

    Se abre la ventana principal **Analizador de Llamadas Pro v2.5** con el semáforo de estado en gris (esperando datos).
  </Step>

  <Step title="Cargar la sábana de llamadas">
    1. Haz clic en **Seleccionar Archivo** y elige tu archivo Excel (`.xlsx`, `.xls`) o CSV.
    2. Si las columnas del archivo no coinciden con el esquema estándar, aparece el **Asistente de Mapeo de Columnas**. Relaciona cada columna del archivo con su campo correspondiente (Originador, Receptor, Duración, Fecha, etc.).
    3. Si el sistema detecta registros sin coordenadas GPS y el archivo tiene una columna de nombre de celda, te preguntará si deseas inferir la ubicación por nombre de municipio. Acepta para activar el Plan C de geocodificación.
    4. Al finalizar, el semáforo cambia a verde y aparece un resumen con el total de registros, entrantes, salientes y datos de internet detectados.
  </Step>

  <Step title="Enriquecer los datos (opcional)">
    Antes de generar el informe puedes añadir contexto:

    - **Asignar Nombres:** Haz clic en **Asignar Nombres/Alias** para poner un nombre legible a los números clave. Los gráficos y tablas del informe usarán esos alias.
    - **Datos del Caso:** Haz clic en **Datos del Caso** para ingresar campos como Cliente, Ciudad, Teléfono, Caso y Período. Estos aparecen en el encabezado del informe.
    - **Adjuntos PDF:** Usa el botón **Agregar** para adjuntar PDFs categorizados (Financiero, Antecedentes, Vehículos, Judicial, Propiedades, Otros).
  </Step>

  <Step title="Generar el informe">
    1. En el campo **Nombre Carpeta**, escribe el nombre de la carpeta de salida (por ejemplo: `Caso_2024_Lopez`).
    2. Marca **Incluir Membretes** si quieres logos corporativos en el informe.
    3. Marca **Subir a FTP** si quieres publicarlo en línea (requiere `.env` configurado).
    4. Haz clic en **Generar y Exportar Informe**.

    El sistema procesa los datos, genera los gráficos, construye los tres mapas y ensambla el HTML. Al terminar, te ofrece abrir el archivo directamente:

    ```
    output/
    └── Caso_2024_Lopez/
        ├── reports/
        │   └── informe_llamadas.html   ← Abre este archivo
        ├── graphics/
        ├── maps/
        └── data/
    ```
  </Step>
</Steps>

---

## Resultado esperado

Al abrir `informe_llamadas.html` en el navegador verás:

- **Dashboard** con gráficos de líneas de tendencia horaria (animados)
- **Top 10** de números más frecuentes (entrantes y salientes)
- **Gráficos de ubicación** que cruzan números con antenas
- **Tres mapas interactivos** en pestañas: clusters, rutas cronológicas y mapa de calor
- **Visor de PDF** para cada adjunto categorizado

<Tip>
  El informe HTML es completamente portable: puedes enviarlo por correo o copiarlo a una USB sin necesidad de conexión a internet ni servidor.
</Tip>
